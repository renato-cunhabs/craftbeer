Craftbeer
----

# Dependências do projeto
* Java 1.8
* Maven
* Docke
* Docker Compose

# Dependências externas

* Postgres

# Executando local

	- executar o docker-compose a partir da pasta deps do projeto.
			- docker-compose up -d

	- Rodar a aplicação:
        rodar a classe /craftbeer/src/main/java/com/beerhouse/Application.java
